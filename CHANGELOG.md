# 更新日誌 (Changelog)

所有重要的版本更新都會記錄在此文件中。

---

## [1.0.0] - 2025-11-06

### 🎉 初始版本發佈

#### ✨ 核心功能
- **用戶認證系統**
  - Email/Password 註冊與登入
  - 用戶資料儲存於 Firestore
  - 暱稱（displayName）管理

- **活動管理**
  - 發起揪團活動（含活動序號累進）
  - 編輯活動（日期、配速、距離、路線、提醒）
  - 刪除活動
  - 參加活動
  - 取消參加活動

- **首頁功能**
  - 橫向日期選擇器（今天 + 7 天）
  - 週末標示（六綠、日紅）
  - 有活動日期顯示跑步圖示 🏃
  - 日期/全部活動切換檢視
  - 即時活動卡片列表

- **我的活動頁面**
  - 我發起的活動列表
  - 我參加的活動列表
  - 編輯/刪除功能按鈕
  - 活動統計顯示

- **活動卡片**
  - 揪團場次編號
  - 日期時間顯示
  - 主揪人資訊
  - 配速、距離、路線
  - 參加人數（可展開/收合）
  - 其他提醒事項
  - 加入/取消按鈕

#### 🎨 UI/UX 設計
- 📱 Mobile-first 響應式設計
- 🎨 使用 Tailwind CSS
- 🔵 主色調為藍色系
- 📍 底部導航列（首頁、我要揪團、我的活動）
- ✨ 流暢的過渡動畫效果

#### 🔧 技術架構
- **前端**: React 18 + Vite + React Router
- **樣式**: Tailwind CSS + PostCSS
- **後端/資料庫**: Firebase (Auth + Firestore)
- **日期處理**: date-fns
- **部署**: Netlify (SPA 路由設定)

#### 🔐 安全性
- Firestore Security Rules 完整設定
- 用戶權限驗證（創建、編輯、刪除）
- 參與者陣列更新權限控管

#### 📋 已知問題修正
- ✅ 修正 SPA 重新整理 404 問題
- ✅ 修正無法取消參加活動的問題
- ✅ 完善編輯活動功能

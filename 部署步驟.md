# 🚀 超簡單部署步驟

## 方法一：自動部署（最簡單） ⭐

如果您的 Netlify 已經連結 GitHub，只要推送程式碼就會自動部署：

### 步驟：

```bash
# 1. 查看修改了什麼
git status

# 2. 將所有修改加入
git add .

# 3. 提交修改（可以改成您想要的訊息）
git commit -m "更新網站"

# 4. 推送到 GitHub（Netlify 會自動部署）
git push
```

✅ **完成！** 等待 2-3 分鐘，Netlify 會自動建置和部署。

---

## 方法二：手動部署

如果方法一不行，可以用 Netlify CLI：

```bash
# 1. 建置網站
npm run build

# 2. 部署到 Netlify
npx netlify-cli deploy --prod
```

第一次使用會要求登入和選擇網站。

---

## 📱 查看部署狀態

### 方式 1：去 Netlify 網站看
1. 打開 https://app.netlify.com/
2. 選擇您的網站
3. 查看「Deploys」頁籤
4. 看到綠色勾勾 ✅ 就是成功了

### 方式 2：用指令查看
```bash
npx netlify-cli status
```

## ⚠️ 常見問題

### Q: 推送後沒有自動部署？
**A**: 檢查 Netlify Dashboard → Site settings → Build & deploy → Build hooks
確認「Auto publishing」是開啟的

### Q: 部署失敗？
**A**: 
1. 到 Netlify Dashboard 查看錯誤訊息
2. 確認環境變數都有設定好
3. 檢查 `netlify.toml` 設定是否正確

### Q: 本地可以跑但部署後不行？
**A**: 
1. 檢查 Netlify 環境變數是否都設定好（共 13 個）
2. 參考 `NETLIFY_QUICK_SETUP.md`
